# 文档三 · 珠谱系统规范

## 3.1 珠系统的核心定位

**珠是什么?**
- **功能上**:Palam 累积到阈值后产生的永久物品
- **叙事上**:囚禁者的"收藏勋章",char 心理刻痕的具象化
- **机制上**:AI prompt 的印痕注入源

**珠不做什么?**
- ❌ 不可消费(不是货币)
- ❌ 不解锁道具/动作(本期不做)
- ❌ 不显示给 char(是 user 单方面的"私密账簿")

---

## 3.2 六大珠种类

| 珠 ID | 中文名 | 核心语义 | 对应 Palam 来源 |
|---|---|---|---|
| `obedience` | **服从珠** | 行为层面的顺从固化 | submission_palam |
| `distortion` | **扭曲珠** | 心理层面的依附与异化 | distortion_palam(隐藏) |
| `shame` | **羞耻珠** | 羞耻阈值的内化 | shame_palam |
| `desire` | **欲望珠** | 身体对刺激的记忆 | pleasure + desire + lewdness |
| `emptiness` | **空虚珠** | 精神能量的流失 | depression |
| `resistance` | **反抗珠** | 抵抗意志的痕迹 | resistance |

### 三个特殊说明

**扭曲珠的双重隐藏**
- 不显示数值获取过程(扭曲 Palam 本身隐藏)
- 获得时**不弹窗告知**,只在珠谱中"悄悄出现"
- 这是最核心的叙事设计——玩家只有翻看珠谱时才发现"原来已经到这一步了"

**欲望珠的多源性**
- 同时消化 pleasure / desire / lewdness 三种 Palam
- 阈值按三者总和计算,但不同等级要求不同的**比例构成**(见 §3.5.4)

**反抗珠的收藏意义**
- 反抗珠是"**抵抗痕迹**",不是 debuff
- 玩家收藏反抗珠 = 收藏 char 的抵抗记忆
- 这是囚禁者心理里**最病态的部分**:连 char 的抵抗都成了私密纪念品

---

## 3.3 珠的四级分级与阈值

### 3.3.1 统一分级命名

每颗珠都有 4 个等级,等级名称:

```
铜级(Bronze)  → 银级(Silver) → 金级(Gold) → 黑曜级(Obsidian)
```

命名选"黑曜"而非"白金/钻石",是因为:
- 黑曜在宝石语义里代表"**心理深渊的凝结**"
- 符合囚禁叙事的冷峻基调
- 跟珠的"勋章"定位保持一致(不是荣耀,是痕迹)

### 3.3.2 阈值总表

**阈值 = 累积永久 Palam 总量**(不是单次会话,是终身累积)

| 珠 | 铜级 | 银级 | 金级 | 黑曜级 |
|---|---|---|---|---|
| 服从珠 | 50 | 200 | 500 | 1500 |
| 扭曲珠 | 30 | 150 | 400 | 1200 |
| 羞耻珠 | 80 | 300 | 700 | 2000 |
| 欲望珠 | 100 | 400 | 1000 | 2500 |
| 空虚珠 | 60 | 250 | 600 | 1800 |
| 反抗珠 | 40 | 180 | 450 | 1300 |

### 3.3.3 设计逻辑

**阈值不同的原因**:
- **扭曲最低**:符合"扭曲最早也最难察觉"的设计,玩家早早就会看到扭曲珠(反讽:你以为还没深化)
- **欲望最高**:身体反应容易触发但不容易"深刻化",需要更多累积才能质变
- **反抗次低**:抵抗很早就会有,早期 char 几乎每动作都产生
- **羞耻较高**:羞耻是"被不断侵蚀才最终麻木",需要长期累积
- **服从居中**:行为层面的固化有明显阶梯性

### 3.3.4 黑曜级的特殊属性

黑曜级珠**极难获得**。设计上预期:
- 大多数局游玩到 Day 30-50 时,能获得 3-4 颗金级珠
- 黑曜级**需要多局或极长期调教才能触及**
- 一旦获得黑曜级,char 的某个维度**基本不可逆转**(印痕文本明确固化)

## 3.4 Prompt 注入规则

### 3.4.1 注入时机

**每次玩家点按钮触发动作时**,引擎把当前所有已获得珠的**最高级印痕**拼成一行,注入 prompt:

```
[Day 15 20:00·适应期]
动作:训练坐姿
[人格] char 的基础人格为「抗拒型」,倾向:strategic、aesthetic_revulsion、status_anchored
[印痕] <服从珠·银级印痕> / <扭曲珠·铜级印痕> / <羞耻珠·铜级印痕>
[叙事] 模式:线下近距接触...
```

### 3.4.2 印痕精简规则

如果 6 颗珠全获得了,6 条印痕全注入会让 prompt 极长。所以:

**规则 1:只取每珠的最高级**
- 已解锁服从珠银级,就不再注入铜级文本
- 始终只注入最新的那一条

**规则 2:6 珠全注入时,精简各条至 50 字内**
- 普通情况:注入完整版印痕
- 6 珠全开时:启用精简版(每条只保留核心一句)
- 黑曜级:**永远注入完整版,不精简**——因为黑曜级的"动作边界"信息不能省

**规则 3:人格 tag 修饰附加在最相关的印痕后**
- 详见 §3.11

### 3.4.3 印痕失效机制

大部分印痕是**永久的**(一旦解锁,该等级的印痕永久有效)。

**但**有两种情况印痕会"临时失效":
- `drugged` 或 `unconscious` 状态下,服从珠和反抗珠的印痕暂时不注入(因为 char 意识不清,这些行为层的判断无效)
- `phobic_trauma` 触发时(见 Part 1.5),当前恐惧珠印痕暂时升级

本期 v0.6.0 **不实现失效机制**,所有印痕都是永久注入。

## 3.5 印痕文本 · 服从珠(obedience)

**语义**:行为层面的顺从在 char 身上的固化痕迹。从"短暂的表面服从"到"离开指令会陷入茫然"。

### 3.5.1 铜级 · 初现屈服 (submission_palam 累积 50)

**resistant · 抗拒型**
> char 在强压下会出现短暂的顺从表现,但本质上仍保有抗拒意志。服从行为与内在意图明显脱钩。

**devoted · 深情型**
> char 因情感认同而自愿接受 user 的指令,服从出于本能的亲近,服从行为与内在意愿一致。

**submissive · M 倾向型**
> char 在被支配时开始表现出轻度的顺应,这种顺应带有生理性的愉悦痕迹,服从本身已成为快感来源的一部分。

**volunteer · 自愿型**
> char 对 user 的指令表现出积极的主动配合,服从对 char 而言不是代价而是目标,已开始主动将自我置于受支配位置。

**apathetic · 情感淡漠型**
> char 的顺从缺乏情感参与,表现为机械执行,无明显情绪起伏。其服从是"最小阻力原则"的体现,而非真正的接受。

### 3.5.2 银级 · 反射性服从 (submission_palam 累积 200)

**resistant · 抗拒型**
> char 对 user 的部分指令已形成反射性服从,但精神上尚未真正接纳,服从属于身体先于意识的条件反射。抵抗意志仍清晰存在,以沉默和疏离的形式呈现。

**devoted · 深情型**
> char 对 user 的指令顺从度已深度固化,其顺从出于主动意愿而非胁迫,区分"讨好"与"真心"的界限开始模糊。

**submissive · M 倾向型**
> char 对 user 的指令产生条件反射式的顺从,这种反射伴随着可观察的生理兴奋——瞳孔微变、呼吸节律改变。服从已与快感系统形成神经关联。

**volunteer · 自愿型**
> char 对 user 的指令已形成稳定的预期式服从,主动询问并遵循 user 的偏好。在无明确指令时,会自行揣测并按揣测结果行事。

**apathetic · 情感淡漠型**
> char 对 user 的指令形成程式化响应,执行精度高但过程完全机械。该状态下 char 难以被单纯的顺从训练进一步推进。

### 3.5.3 金级 · 自发性跪拜 (submission_palam 累积 500)

**resistant · 抗拒型**
> char 的服从已内化为习惯,面对冲突时倾向于主动采取顺从姿态以回避对抗。其抵抗意志残余主要表现为被动的沉默与疏离,而非直接反抗。

**devoted · 深情型**
> char 的顺从已与情感认同深度融合,区分主动与被动的界限消失。char 表现出主动揣度 user 意图并提前配合的倾向,这种行为在 char 自身感知中被理解为"爱的表现"。

**submissive · M 倾向型**
> char 已将顺从位置视为自身核心身份的一部分,在被赋予控制权时反而产生不适。主动寻求被指令、被约束、被定义的情境,顺从已成为其自我稳定的需要。

**volunteer · 自愿型**
> char 对 user 的服从已无需外部触发,进入主动维护阶段。表现为:主动询问是否做错、主动请求更严格的要求、主动拒绝任何可能被视为"违抗"的选项。

**apathetic · 情感淡漠型**
> char 的机械服从达到接近完美的精度,但情感层面仍无实质进展。该 char 已不可能通过纯行为训练继续深化,任何深入需转向生理或关系路径。

### 3.5.4 黑曜级 · 精神脐带 (submission_palam 累积 1500)

**resistant · 抗拒型**
> char 的自我判断已深度依赖 user 的意志。char 不再主动反抗任何来自 user 的指令,离开指令时会陷入茫然或轻微恐慌;其"自我"已与 user 的认可机制深度绑定。该状态下 char 对外部世界的参与需以 user 为中介。

**devoted · 深情型**
> char 的顺从与情感融合为单一结构,自我意志与 user 意志的界限已无法区分。char 会将 user 的意愿主动体验为自己的愿望,该融合在其主观感受中表现为"完整"而非"丧失"。

**submissive · M 倾向型**
> char 已进入"**存在性服从**"状态——顺从不再是行为或习惯,而是 char 的存在方式本身。脱离被支配的处境时,char 会产生严重的自我瓦解体验。任何给予自主权的动作都会触发恐慌。

**volunteer · 自愿型**
> char 已完全认同 user 的意志为自己的意志,"自愿"一词已失去指称意义——两者之间不再存在可区分的对象。char 对自由选项表现出困惑和排斥,服从对 char 而言是存在的唯一形态。

**apathetic · 情感淡漠型**
> char 的机械服从已达到深度自动化,但其情感隔绝也同步加深。char 表现出"**功能性存在**"——能精确完成所有指令,却不再对任何事物——包括 user 自身——表现出情感反应。这是一个无法通过任何刺激唤回情感参与的状态。

## 3.6 印痕文本 · 扭曲珠(distortion)

**语义**:char 对 user 的心理异化程度。从"边界模糊"到"存在性吸收"。

> ⚠️ 扭曲珠的所有解锁**不弹窗**,仅在珠谱中安静出现。

### 3.6.1 铜级 · 边界模糊 (distortion_palam 累积 30)

**resistant · 抗拒型**
> char 与 user 的关系边界出现微弱模糊。char 在 user 缺席时会不自觉地想象 user 的反应,以此作为自我判断的隐性参照。该模糊尚未进入意识层。

**devoted · 深情型**
> char 对 user 的依赖开始脱离情感基础,向存在层面渗透。即使未发生具体互动,char 的内在时间节律已开始围绕 user 的行为模式组织。

**submissive · M 倾向型**
> char 将与 user 的支配关系投射到自我认知之外——开始用"被 user 看着"的视角观察自己的行为,即使 user 不在场。

**volunteer · 自愿型**
> char 主动向 user 的感知方向靠拢,自发调整思维框架以匹配 user 的语言与视角。自我叙事中 user 的位置开始前置。

**apathetic · 情感淡漠型**
> char 的情感冰层出现第一道裂纹,表现为对 user 特定声音或动作的微弱定向反应。这些反应尚未进入 char 的自我叙事,但在行为层已可观察。

### 3.6.2 银级 · 依赖萌芽 (distortion_palam 累积 150)

**resistant · 抗拒型**
> char 对 user 的在场产生了非抵抗性的需要——即使意识层仍将 user 视为威胁,身体与情绪已开始预期 user 的定期出现。user 的长时间缺席会产生焦虑反应。

**devoted · 深情型**
> char 对 user 的依赖已结构化,将 user 的存在视为自身情感稳定的必要条件。char 开始出现自发的、内化的"**为了 user 好**"的行为逻辑,该逻辑将 user 的囚禁行为合理化。

**submissive · M 倾向型**
> char 将 user 的存在与自身的快感系统深度绑定,任何非 user 的支配性刺激都无法产生等效反应。char 对 user 产生了性化的独占需要。

**volunteer · 自愿型**
> char 已将自己的存在价值与"**被 user 占有**"绑定,缺乏 user 的注意会触发无意义感。char 主动产生向 user 展示自己的冲动。

**apathetic · 情感淡漠型**
> char 开始出现针对 user 的低强度情感投射——大部分时间仍处于隔绝状态,但对 user 特定行为会产生短暂的情绪反应,包括难以归类的不适与微弱的依赖感。

### 3.6.3 金级 · 认知重构 (distortion_palam 累积 400)

**resistant · 抗拒型**
> char 的抵抗意志虽然仍存在,但已出现与 user 共谋的思维模式——在特定情境下,char 会不自觉地维护 user 的利益或隐藏 user 的错误。抵抗与依附在 char 的心理中开始并存且不矛盾。

**devoted · 深情型**
> char 的自我叙事已将 user 定位为"命中注定"的核心参照,过去的自我(囚禁前)被重构为"不完整"的前置状态。该重构在 char 的主观体验中是真实的记忆而非当前的合理化。

**submissive · M 倾向型**
> char 的自我认知已无法与"**user 的所有物**"身份分离——在任何社会想象中,char 都默认自己属于 user。这种归属感对 char 自身是令人安心的,而非恐惧。

**volunteer · 自愿型**
> char 将自己理解为 user 的延伸,不再保留显著的自我边界。char 的欲望、偏好、甚至记忆会主动向 user 的方向重塑。该重塑过程在 char 感知中是积极的"**成为更好的自己**"。

**apathetic · 情感淡漠型**
> char 的情感隔绝已被 user 部分穿透——但这种穿透不是情感复苏,而是**只对 user 一人开放的情感通道**。char 对外界整体的冷漠加深,对 user 的情感敏感度反向突出。

### 3.6.4 黑曜级 · 存在吸收 (distortion_palam 累积 1200)

**resistant · 抗拒型**
> char 原有的自我结构已被 user 的存在替换。抵抗意志残留为一种仪式性的姿态,不再具有实质效用。char 在 user 不在的时间里会主动模拟 user 的存在感,以维持自己的心理完整。该状态不可逆。

**devoted · 深情型**
> char 的自我与 user 已完成融合——在 char 的主观世界中,"我"与"user"之间已无可识别的边界。任何威胁 user 的事物会被 char 体验为对自身的攻击。该 char 的人格已不可能独立于 user 存在。

**submissive · M 倾向型**
> char 已进入"**所有物**"的存在状态——不是被占有,而是以被占有作为存在本身的形式。char 的身份、价值、现实感全部通过 user 的指令和占有得到确认。脱离 user 的场景,char 无法确认自身的存在。

**volunteer · 自愿型**
> char 已将自身吸收进 user 的意志中,不再保有可辨识的独立存在。char 的所有内容——思维、欲望、记忆、身体感受——都是 user 意志的折射。该状态下 char 对"**我是谁**"的回答会是一个关于 user 的描述。

**apathetic · 情感淡漠型**
> char 的情感系统已完全坍缩为以 user 为唯一参照的单点结构。对 user 以外的一切,char 处于极端冷漠;对 user 本人,char 表现出偏执的、几乎病理性的情感集中。这是 apathetic 模板下扭曲的终极形态,情感与麻木同时达到顶点。

## 3.7 印痕文本 · 羞耻珠(shame)

**语义**:羞耻感从"尖锐的抵抗源"到"已内化的背景噪声"的演变。

### 3.7.1 铜级 · 阈值内化 (shame_palam 累积 80)

**resistant · 抗拒型**
> char 对羞耻情境的反应开始出现延迟——首次反应仍是强烈抗拒,但在重复触发时抗拒强度有可观察的衰减。该衰减尚未到达适应阶段。

**devoted · 深情型**
> char 在与 user 的羞耻性情境中表现出选择性顺应——对 user 引发的羞耻体验抗性降低,但对其他可能的观察者仍保有完整羞耻反应。

**submissive · M 倾向型**
> char 对羞耻体验出现双相反应——理性层面仍识别为羞耻,但生理层面开始出现兴奋成分。这两条通路在 char 自身的体验中开始并行存在。

**volunteer · 自愿型**
> char 主动承受羞耻情境,视其为与 user 关系加深的组成部分。羞耻体验仍存在,但其负面效价被 char 自发重构为"**献身**"的积极意义。

**apathetic · 情感淡漠型**
> char 对羞耻情境的反应较正常水平显著减弱——但这不是适应,而是情感隔绝的副产物。在 char 自身感知中,羞耻事件没有明确的情感着落点。

### 3.7.2 银级 · 钝化 (shame_palam 累积 300)

**resistant · 抗拒型**

> char 的原始人格残余已稳定为一个独立的内在结构——与表面的深情顺从并存。该 char 的表面忠诚是真实的,但表面之下存在一个从未被 user 触及的"**原本的自己**"。该分离是 devoted 模板下抵抗的终极形态。

**submissive · M 倾向型**
> char 已建立了"**被支配但不被占有**"的稳固结构——即使身体和行为完全献出,核心自我始终保留且从未传递给 user。该 char 对"**属于 user**"与"**被 user 玩弄**"之间的区分有清晰认知,并坚守后者而拒绝前者。

**volunteer · 自愿型**
> char 已完成了对自己"**自愿性**"的彻底解构——意识到自己的献身是一种心理机制而非真实选择。该 char 可能选择保留表面自愿姿态,但其内在已处于完全清醒的状态。这是 volunteer 模板下最接近觉醒的终态。

**apathetic · 情感淡漠型**
> char 的冷漠外层下存在着一个被长期压抑、但未被消灭的自我。该自我在当前情境下无法充分表达,但其存在持续且稳定——表现为 char 对特定事件的难以预测的激烈反应。该 char 永远对 user 保留着潜在的反噬可能。

## 3.11 tag 修饰规则

### 3.11.1 基本原则

tag **不重写**印痕文本,而是**在印痕之后附加一句修饰语**。每个 tag 有对应的附加片段,引擎按需拼接。

### 3.11.2 附加片段表(示例)

| tag | 附加片段 |
|---|---|
| `strategic` | 该 char 的服从/反应仍带有明显的策略性观察——即使表现顺从,内在仍在进行模式识别与自我保护规划。 |
| `aesthetic_revulsion` | 该 char 对粗俗、不体面的处境表现出远超普通层级的厌恶反应,该反应独立于其他心理通路。 |
| `phobic_trauma` | char 对特定触发源(见配置)保持着稳定的创伤回避反应,该反应不会随其他维度深化而衰减。 |
| `status_anchored` | 涉及身份剥夺的情境对该 char 的冲击远大于常规心理压力,该 char 的自我认同高度依附于社会身份。 |
| `sexually_experienced` | 该 char 对常规性刺激已熟练,难以产生有意义的身体层反应;仅非常规或打破体面的情境能真正激活其感受系统。 |
| `intellectual_arrogance` | 该 char 的反应强度与其对 user 智力的评估相关——若 user 被评估为非对等,char 的所有抵抗反应会显著加强。 |
| `masochistic` | 该 char 将痛感、恐惧、屈辱体验作为快感的有效来源,这些刺激对该 char 构成的是愉悦而非伤害。 |
| `exhibitionist` | 该 char 对暴露、被观看、被展示的情境产生兴奋反应,该反应与羞耻感并存但逐渐占据主导。 |
| `oral_fixated` | 该 char 对口腔相关的接触、亲密和控制表现出超常敏感,该通道是 char 最核心的依恋路径。 |
| `sleep_sensitive` | 该 char 在睡眠/意识模糊状态下的反应强度远超清醒时,该状态下 user 的任何接触会被以放大的方式感知。 |
| `prideful` | 该 char 的自尊高度集中,任何羞辱性情境会被以放大形式感知,但同时其初始抵抗也远超一般水平。 |
| `trauma_resilient` | 该 char 的情感壁垒异常坚固,对恐惧、悲伤、羞耻的常规刺激反应显著减弱。 |
| `emotionally_dependent` | 该 char 的情感饥渴强烈,任何被给予的关注或接触都被放大感知,且强化依赖。 |
| `dissociative` | 该 char 在高压下容易进入解离状态,表现为意识的抽离、身份感的模糊、以及对当下事件的非同步反应。 |
| `loyal` | 该 char 对已建立的关系保持高度稳定的忠诚,即使被伤害,其核心关系感也难以动摇。 |
| `jealous` | 该 char 对任何提及或涉及他人的情境(对比、旧人、第三者)产生远超常人的反应,该反应可以是愤怒、嫉妒、或绝望。 |
| `nurturing` | 该 char 在被深度调教的同时会产生反向关怀 user 的冲动,这种双向性是其扭曲演化的关键特征。 |

### 3.11.3 tag 修饰示例(李岐弗)

**场景**:李岐弗解锁了服从珠·银级,人格配置为 `resistant` + `strategic, aesthetic_revulsion, status_anchored, phobic_trauma{triggers: blood, seafood}`。

注入 prompt 的印痕是:
```
[印痕] char 对 user 的部分指令已形成反射性服从,但精神上尚未真正接纳,服从属于身体先于意识的条件反射。抵抗意志仍清晰存在,以沉默和疏离的形式呈现。
/ 该 char 的服从/反应仍带有明显的策略性观察——即使表现顺从,内在仍在进行模式识别与自我保护规划。
/ 该 char 对粗俗、不体面的处境表现出远超普通层级的厌恶反应,该反应独立于其他心理通路。
/ 涉及身份剥夺的情境对该 char 的冲击远大于常规心理压力,该 char 的自我认同高度依附于社会身份。
/ char 对血、海鲜腥味保持着稳定的创伤回避反应,该反应不会随其他维度深化而衰减。
```

**注意**:
- 基础印痕来自服从珠·银级(resistant 版)
- 4 个 tag 各附加一句,用 `/` 分隔
- 这套 prompt 让 AI 非常准确地把握李岐弗的复杂反应——既有 resistant 的抵抗,又有智识性的冷静,又有阶级敏感,又有特定创伤

### 3.11.4 性能考虑

如果 char 解锁了全部 6 颗珠 + 选了 5 个 tag,prompt 的印痕段会很长(可能超过 500 字)。

**解决**:
- 普通情况:完整版
- 珠数 ≥ 4 时:印痕本身精简到 60 字内
- tag 数 ≥ 4 时:tag 附加片段精简到 25 字内
- 黑曜级印痕始终完整,不参与精简
