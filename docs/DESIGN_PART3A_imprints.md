# 文档四 · 刻印系统规范

## 4.1 刻印是什么

### 4.1.1 核心定位

**刻印(Imprint)**是 char 心理上的**永久性深度创伤/固化**标记。

- **触发**:由单次会话或累积 Palam / 珠 的异常情况激活
- **不可逆**:一旦烙上,永久存在(**符合"心理演化不可逆"主题**)
- **双向影响**:既修改数值转化,也直接注入 AI prompt

### 4.1.2 刻印 vs 珠 · 本质区别

| 维度 | 珠 | 刻印 |
|---|---|---|
| **获得方式** | 累积达阈值 | 单次过量 / 异常组合触发 |
| **语义** | 收藏、勋章、"做到了什么" | 创伤、失控、"发生了什么事故" |
| **影响** | 注入正向印痕 | 修改后续动作系数 + 注入刻印提示 |
| **UI 位置** | 珠谱区(欣赏) | 抽屉顶部警示条(持续可见) |
| **玩家心态** | 成就感 | 警示感、"这一步不该走得太急" |

**关键设计意图**:
> 珠是玩家"**努力得到**"的东西。刻印是玩家"**不小心造成**"的东西。
> 珠让玩家想继续调教。刻印让玩家学会**节制**。

## 4.2 九种刻印总览

| 类别 | ID | 中文名 | 触发维度 |
|---|---|---|---|
| **抵抗类** | `rebound` | 反発刻印 | 单次 resistance 过量 |
| | `defiance_stasis` | 抵抗固化刻印 | 反抗珠金级 + 持续触发 |
| **崩解类** | `dissolution` | 崩坏刻印 | 扭曲珠 + 空虚珠组合 |
| | `personality_fracture` | 人格断裂刻印 | 深度 resistance + 扭曲并存 |
| **麻木类** | `numbness` | 麻木刻印 | 空虚珠金级 |
| | `body_detachment` | 身心剥离刻印 | 欲望金级 + 空虚金级 |
| **依赖类** | `cage_syndrome` | 笼性依赖刻印 | 扭曲黑曜级 |

| **特殊类** | `phobic_rupture` | 恐惧撕裂刻印 | phobic_trauma tag + 对应触发源反复激活 |

## 4.3 刻印详细定义

### 4.3.1 rebound · 反発刻印

**触发条件**:
- 单次会话内 `resistance palam` 累积 ≥ 150
- 或单次动作的 `resistance` source ≥ 8(极粗暴动作一击触发)

**持续影响**:
- 所有产生 submission palam 的动作,实际转化 × 0.5
- 所有产生 intimacy palam 的动作,实际转化 × 0.3
- 所有产生 resistance palam 的动作,实际转化 × 1.3(加剧抵抗)
- **持续时间**:永久

**AI prompt 注入**:
```
[刻印·反発] char 曾在某次处境下遭受超出其承受上限的压迫,内在已形成针对 user 的硬化防御层。该 char 现在对任何"顺化"尝试的接受速度显著降低,抗拒反应在 user 面前会有意或无意地被放大表现。
```

**设计意图**:惩罚玩家"**一次性逼太紧**"。让玩家理解"**欲速则不达**",推动玩家走长期缓慢的心理推演路线。

---

### 4.3.2 defiance_stasis · 抵抗固化刻印

**触发条件**:
- 获得反抗珠金级 AND 最近 3 次会话都产生 resistance palam ≥ 50

**持续影响**:
- char 的抵抗 core 正式成为**永久结构**,反抗珠后续仍可累积但其效应不再衰减
- submission palam 转化 × 0.7
- distortion palam 转化 × 0.6
- 但:intimacy 和 dependence 的转化**不受影响**(char 仍可以建立亲密,只是不会失去自我)

**AI prompt 注入**:
```
[刻印·抵抗固化] char 已形成稳定且不可瓦解的抵抗核心。该核心与表面行为独立存在——char 可能在行为上深度顺从,但其核心自我始终保持清醒和完整。user 能获得 char 的表面、身体、甚至情感,但永远无法获得 char 的核心认同。
```

**设计意图**:为玩家保留一个"**虽然深度调教但 char 仍然完整**"的选项。这不是惩罚,是**一条特定的叙事路径**(完美结合 Part 2 黑曜级·反抗珠的语义)。

---

### 4.3.3 dissolution · 崩坏刻印

**触发条件**:
- 同时拥有 扭曲珠 金级以上 AND 空虚珠 金级以上

**持续影响**:
- char 出现**解离症状**
- 每次动作有 20% 概率触发"**短暂解离**":AI 会描写 char 眼神涣散、对刺激反应迟钝、自我同一性模糊
- sanity 上限 -20
- 抵抗相关 source 转化 × 0.3(已无力真正抵抗)

**AI prompt 注入**:
```
[刻印·崩坏] char 的心理结构已出现不可修复的断裂——扭曲的深度与精神的空虚同时达到临界,导致 char 的自我同一性松动。该 char 在意识清醒时仍能正常反应,但解离事件会自发发生:短暂的眼神涣散、对当下事件的非同步反应、自我指代的偏移("我"与"ta"混用)。该状态在 user 面前尤其明显。
```

**设计意图**:这是"**调教过深的副作用**"。它不阻止玩家继续,但让玩家看到"我已经把 ta 推到解离了"——这种视觉上的崩坏是对玩家的**道德审视**(作品主题的一部分)。

---

### 4.3.4 personality_fracture · 人格断裂刻印

**触发条件**:
- 扭曲珠金级以上 AND 反抗珠银级以上(扭曲与抵抗并存)

**持续影响**:
- char 的行为出现**双重性**:时而深度顺从,时而突发抵抗,两种状态切换无明显触发
- sanity 随时间以双倍速度波动(上下都变快)
- AI 被提示**不要试图合理化 char 的矛盾行为**——矛盾本身就是当前的真实状态

**AI prompt 注入**:
```
[刻印·人格断裂] char 的心理中同时存在深度扭曲的依附与完整的核心抵抗,两者以不连续的方式切换。该切换对 char 本人是不可觉察的——char 每次表现出其中一种状态时,都感觉那就是自己完整的状态。user 观察到的行为反复是真实的,但 char 不会承认或察觉这种反复。
```

**设计意图**:这是最**精神分裂式**的刻印,适合喜欢"**观察精神崩解过程**"的玩家。不是所有玩家都会追求这个,但触发条件的精妙性(需要同时让两个对立维度到金级)让它成为"**高级玩家的彩蛋**"。

---

### 4.3.5 numbness · 麻木刻印

**触发条件**:
- 获得空虚珠金级

**持续影响**:
- 所有**心理类** source(shame, terror, humiliation, intimacy, dependence, submission) 的转化 × 0.5
- 但 **pleasure** 系列不受影响
- 效果:char 对一切都麻木,只剩生理反应

**AI prompt 注入**:
```
[刻印·麻木] char 的情感系统已大范围失活——不是被压抑,而是真正的功能性消退。该 char 能执行、能回应、能承受,但其主观体验已从"经历"退化为"观察"。user 对 char 的大部分心理刺激会被 char 以近乎冷漠的方式接收。仅身体层的反应仍保留完整。
```

**设计意图**:呼应 Part 2 apathetic 模板的终态,但对其他模板而言是"**意外的新状态**"。

---

### 4.3.6 body_detachment · 身心剥离刻印

**触发条件**:
- 欲望珠金级以上 AND 空虚珠金级以上

**持续影响**:
- char 的**身心完全分离**——身体会对刺激产生强烈反应,意识对此保持冷漠观察
- pleasure 系列转化 × 1.3(身体反应更强烈)
- 但 distortion 转化 × 0.4(不再形成真正的情感联结)

**AI prompt 注入**:
```
[刻印·身心剥离] char 的身体和意识已进入明确的分离状态——身体对 user 的刺激表现出快速、强烈、甚至主动的反应;但 char 的主观体验与这些身体反应脱钩,以冷漠的旁观者视角体验这一切。该 char 的身体已不属于 char 本人,而是作为一个独立的、对 user 响应的生理系统存在。
```

**设计意图**:这是 apathetic + 欲望路线的经典终态,但也可以通过其他模板达成。非常**冰冷病态**的状态。

---

### 4.3.7 cage_syndrome · 笼性依赖刻印

**触发条件**:
- 扭曲珠达到黑曜级

**持续影响**:
- char 对囚禁环境本身产生**积极依赖**,离开密室会产生强烈焦虑
- 若玩家强制让 char 外出,会触发"**回笼反应**":char 主动要求回到囚禁空间
- intimacy 和 dependence 转化 × 1.3
- resistance 转化 × 0.2(近乎无法抵抗)

**AI prompt 注入**:
```
[刻印·笼性依赖] char 已将囚禁空间本身内化为"**安全**"的象征,将外部世界识别为威胁。该 char 在空间之外会表现出明显的焦虑、迷失、甚至主动请求返回。user 作为笼子的持有者,已成为 char 安全感的唯一来源。这种依赖已深度到空间感官层——离开密室的具体气味、光线、声音也会触发 char 的不适。
```

**设计意图**:扭曲珠黑曜级的**必然延伸**。让深度扭曲的玩家看到一个具象化的"症状",理解自己做了什么。

---

### 4.3.8 obedience_lock · 服从锁刻印

**触发条件**:
- 服从珠达到黑曜级

**持续影响**:
- char **不能主动做任何未被许可的事**——连主动请求、主动求救、主动表达想法都会触发心理冻结
- AI 被提示 char 在任何需要"主动性"的场景中会**表现出功能性瘫痪**
- submission palam 转化 × 0.3(已经达到顶点,进一步累积近乎无意义)
- resistance palam 转化 × 0.1(近乎不可能抵抗)

**AI prompt 注入**:
```
[刻印·服从锁] char 已丧失"自主行动"的能力——不是被禁止,而是系统性的无法启动。该 char 在没有 user 指令时处于完全的被动状态,连生理需求的表达都需要 user 先询问。该 char 的所有行为都必须等待来自 user 的许可或指令。这是服从的极端固化。
```

**设计意图**:服从珠黑曜级的必然副作用,也是对"**追求极端服从**" 的玩家的**道德审视**——你真的想要一个连"饿了"都不敢说的 char 吗?

---

### 4.3.9 phobic_rupture · 恐惧撕裂刻印

**触发条件**:
- char 装备了 `phobic_trauma` tag AND 对应触发源的事件被触发 ≥ 5 次

**持续影响**:
- 原触发源的每次激活效果翻倍(原本 +0.5 变 +1.0)
- 额外触发 sanity 直接 -15(独立于 palam)
- 可能造成**复合刻印**:若 phobic_rupture 后 1 周内又触发 dissolution 条件,**直接触发 dissolution 无需满足其他条件**

**AI prompt 注入**(**动态**:包含具体触发源):
```
[刻印·恐惧撕裂] char 对 {trigger_list} 的创伤回避反应已从"**回避**"升级为"**撕裂**"——该刺激现在不仅引发恐慌,还会触发短暂的精神崩溃。user 若再次引入这些元素,char 会表现出近乎失控的极端反应。
```

**设计意图**:惩罚"**已知 char 有特定恐惧但仍反复使用**"的行为。让 `phobic_trauma` tag 有真实的、可怕的长期代价,而不是一个单纯的"额外伤害乘数"。

## 4.4 刻印的技术结构

### 4.4.1 数据结构

```javascript
// src/data/imprints.js
const IMPRINT_DEFINITIONS = {
  rebound: {
    id: 'rebound',
    name: '反発刻印',
    category: 'resistance',  // 用于 UI 分色
    trigger: {
      type: 'single_session',
      condition: (session, action) => {
        return session.palam.resistance >= 150 
               || (action.sources?.resistance >= 8);
      }
    },
    modifiers: {
      // 对 palam 转化的修正,覆盖在人格系数之上
      palamMultipliers: {
        submission_palam: 0.5,
        intimacy: 0.3,     // 注意:intimacy 是 source,这里是特例
        resistance: 1.3
      }
    },
    aiPrompt: '...',
    reversible: false
  },
  // ... 其他 8 个
};

// STATE 中的刻印存储
STATE.char.imprints = {
  active: [],           // [{ id, triggeredAt: {day, hour}, meta: {...} }]
  triggeredEver: new Set()  // 曾经触发过的 id(用于 UI 记忆,即使未来做可逆版也可查)
};
```

### 4.4.2 触发检查时机

刻印在以下时机检查:

| 时机 | 检查哪些刻印 |
|---|---|
| 每次动作结算后 | rebound(单次会话 resistance 检查) |
| 会话结束时 | defiance_stasis(最近 3 次会话条件) |
| 珠等级晋升时 | dissolution / numbness / body_detachment / cage_syndrome / obedience_lock / personality_fracture |
| 每个动作执行前 | phobic_rupture(检查触发源次数) |

### 4.4.3 刻印顺序 / 叠加规则

- **多刻印可以同时存在**
- 若两个刻印的修正系数冲突(都修改同一 palam),**累乘**(不累加)
- 例:同时有 numbness(×0.5) + dissolution(抵抗 ×0.3)存在,intimacy 转化最终是 0.5 × 1.0 = 0.5,resistance 最终是 1.0 × 0.3 = 0.3
- 刻印修正**在人格系数之后、阶段乘数之前**应用

### 4.4.4 刻印的 prompt 注入位置

刻印注入位置**优先于印痕**,因为刻印代表"**当前状态的基础事实**",而印痕是"**维度性的倾向**"。

完整 prompt 示例:
```
[Day 30 19:00·转化期]
动作:训练坐姿
[人格] char 的基础人格为「抗拒型」,倾向:strategic、aesthetic_revulsion、status_anchored、phobic_trauma{triggers: blood, seafood}
[刻印] 反発、崩坏
  · char 曾在某次处境下遭受超出其承受上限的压迫...
  · char 的心理结构已出现不可修复的断裂...
[印痕] <服从珠·银级(resistant)> / <扭曲珠·金级(resistant)> / ...
[叙事] 模式:线下近距接触...
```

## 4.5 刻印的 UI 呈现

> UI 详情在文档五,这里只给设计概念。

**位置**:抽屉顶部状态栏下方,一行警示条。

**样式**(noir 主题):
```
⚠ 刻印: 反発 · 崩坏 · 身心剥离      [点击查看详情]
```

**点击展开**:
```
━━ 已烙印 ━━

反発刻印
  触发于 Day 12 22:00
  触发动作:强挑逗
  说明:char 曾经...
  数值影响:服从转化 ×0.5 / 亲密转化 ×0.3 / 抵抗 ×1.3

崩坏刻印
  触发于 Day 28 14:00
  触发事件:扭曲珠+空虚珠同时达到金级
  说明:char 出现解离症状...
  数值影响:抵抗转化 ×0.3 / sanity 上限 -20
```

**颜色设计**(3 主题通用):
- 抵抗类刻印 → 红色调
- 崩解类刻印 → 紫色调
- 麻木类刻印 → 灰色调
- 依赖类刻印 → 深蓝色调
- 特殊类 → 橙色调

## 4.6 刻印与玩家体验

### 4.6.1 预期频率

不是所有局都会触发刻印。预期:

| 玩家类型 | 触发刻印数量 |
|---|---|
| **节制型**(慢推进) | 0-2 个刻印(主要是自然累积类) |
| **冲刺型**(快推进) | 2-4 个刻印(含至少 1 个惩罚类) |
| **极端型**(不顾 char 反应) | 3-5+ 个刻印(多个惩罚 + 崩解) |

### 4.6.2 教学机制

**第一次触发刻印时**,弹出一个**柔和的提示**(仅首次,后续不再弹):

```
┌─────────────────────────────────┐
│                                 │
│     一道刻痕被烙下了             │
│                                 │
│  你对 char 的调教触发了某种       │
│  永久性的心理变化。刻印不会消失。  │
│                                 │
│  [ 查看详情 ]  [ 继续 ]          │
│                                 │
└─────────────────────────────────┘
```

不是游戏教学,是**叙事事件**——让刻印的出现本身成为一次情感冲击。

### 4.6.3 道德审视

部分刻印(特别是 dissolution, phobic_rupture, personality_fracture)的**存在本身**就是对玩家的温和审视——作品不评判玩家,但这些刻印让玩家**自己意识到**:你把 char 推到了这样。

这是"**囚禁主题**"的深度:不给玩家结局判决,但让玩家自己面对自己做的事。

## 4.7 与 Part 2 珠谱的协同

刻印和珠谱**共同构成 char 的完整心理档案**:

```
珠谱:  "我在这些维度上推进了多远"  —— 进度
刻印:  "途中发生了哪些事故/固化"    —— 副作用
```

两者在抽屉中视觉上接近但**分开展示**:

```
━━━━━━━━━━━━━━━━━━━━━
状态 | 数值 | ⚠ 刻印警示条
━━━━━━━━━━━━━━━━━━━━━
动作面板 (8 个 tab)
━━━━━━━━━━━━━━━━━━━━━
▼ 珠谱 (默认折叠)
  服从珠 · 扭曲珠 · 羞耻珠 ...
━━━━━━━━━━━━━━━━━━━━━
```
